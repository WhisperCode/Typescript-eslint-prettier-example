# Typescript-eslint-prettier-example

## 安装eslint,prettier的开发依赖

### yarn 安装方式

```shell

yarn add eslint prettier @typescript-eslint/eslint-plugin @typescript-eslint/parser eslint-config-prettier 
eslint-plugin-prettier -D
```

###  npm 安装方式

```shell
yarn add eslint prettier @typescript-eslint/eslint-plugin @typescript-eslint/parser eslint-config-prettier 
eslint-plugin-prettier -D
```
